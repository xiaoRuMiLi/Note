weixin-js-sdk 接入微信分享

简介

微信分享没有自动分享功能，实现后只能在微信打开的网页环境使用，还是只能通过浏览器页面右上角的原生的分享按钮分享。只不过能配置分享的标题，分享的缩略图。分享描述。

准备工作

1. 申请微信公众号，订阅号或者服务号。分享必须通过认证，大概花费300元。
2. 将服务器后端域名域名加入IP白名单。 将服务器前端解析过的域名加入js接口安全域名
3. 将weixinJssdk.php 文件上传到服务器后端并编写接口
4. 编写前端代码

常见报错

现将调试中遇到的问题做一个记录，

realAuthUrl

抱着个错是因为后端参与签名的URL和当前页面的URL不一致，意思是要将单前页面路径传给后端进行签名，signature的获取。

invalid signature 签名无效

获取签名的后端服务器未配置证书，当然也许还有很多其他的问题也会造成一样的问题，但是我配置好证书后就解决了.

分享出来任然只是一个链接的路径文字没有任何分享的效果

可以试一下从电脑微信端打开分享出来就是有效果的链接，然后手机端打开这个链接再分享出去就是有效果的。意思是不支持从文字链接分享成有效果的链接
