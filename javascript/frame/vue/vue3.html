响应始对象

reactive

对reactive 中的某个属性为数组的响应式。 如果丢失响应的话，可以这样赋值， 

    
        const state = reactive({
         arr : [1,2]
        })

        const handle = function() {
             const newArr = [2,3];
            // 赋值后失去了响应，没有渲染页面
            state.arr = newArr ;
            // 可以采用下面的方法
            state.arr.length = 0;
            state.arr.push(2);
            state.arr.push(3);

        }

因为state.arr 是一个相依式对象 被赋值为newArr后，newArr是非响应式对象，所以失去了响应
